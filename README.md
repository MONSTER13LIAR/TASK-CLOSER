## ğŸ“– How to Use

### Managing Tasks

1. **Add a Task**: Type in the input field at the top and press Enter
2. **Complete a Task**: Click the circle icon next to the task
3. **Edit a Task**: Click the edit icon to rename
4. **Reorder Tasks**: Drag and drop tasks to change priority
5. **Delete a Task**: Click the trash icon to remove

### Using the Calendar

1. **Add Events**: Click any date to create a new event
2. **Reschedule**: Drag event dots between dates
3. **Remove Events**: Drag events to the red trash bin
4. **Navigate**: Use arrow buttons to move between months
5. **View Upcoming**: Scroll the event list below the calendar

### Setting Moonshot Goals

1. Navigate to the **Goals** page
2. Enter your goal title, duration (in years), and priority level
3. Press **COMMIT** to save permanently
4. Track progress and delete when achieved

### Tracking Your Progress

1. Visit your **Profile** to see productivity statistics
2. Complete all daily tasks to earn a **Perfect Run**
3. Accumulate perfect runs to **Level Up** your mastery
4. Monitor your overall task completion rate
